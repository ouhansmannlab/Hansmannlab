This directory contains initial pdb structures and parameter files used in monomer simulations.

RC1_initial : Initial structure of amyloid β replica 1 control simulation
RC2_initial : Initial structure of amyloid β replica 2 control simulation
RC3_initial : Initial structure of amyloid β replica 3 control simulation

RF1_initial : Initial structure of amyloid β replica 1 with FI10 simulation
RF2_initial : Initial structure of amyloid β replica 2 with FI10 simulation
RF3_initial : Initial structure of amyloid β replica 3 with FI10 simulation



minim.mdp ; Parameter file for energy minimization
nvt.mdp   ; Parameter file for NVT equilibration
npt.mdp   ; Parameter file for NPT equilibration
md.mdp 	  ; Parameter file for production MD run
